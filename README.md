## 📋 Giới thiệu | Overview

Website thương mại điện tử chuyên nghiệp cho **Công ty Cổ phần 379 TDK**, chuyên cung cấp thiết bị vệ sinh và dân dụng cao cấp tại Hưng Yên.

Professional e-commerce website for **379 TDK Joint Stock Company**, specializing in premium sanitary and household equipment in Hung Yen, Vietnam.

## 📦 Cách đưa lên GitHub | Deploy to GitHub

### Bước 1 – Tạo Repository

```bash
# Đăng nhập GitHub, tạo repo mới tên: tdk379-website
# Hoặc dùng GitHub CLI:
gh repo create tdk379-website --public --description "379 TDK - Thiết Bị Vệ Sinh Cao Cấp"
```

### Bước 2 – Clone & Copy file

```bash
git clone https://github.com/YOUR_USERNAME/tdk379-website.git
cd tdk379-website

# Copy tất cả file từ thư mục tdk379/ vào đây
cp -r /path/to/tdk379/* .
```

### Bước 3 – Commit & Push

```bash
git add .
git commit -m "🚀 Initial commit – 379 TDK Website"
git push origin main
```

### Bước 4 – Bật GitHub Pages

1. Vào **Settings** → **Pages**
2. Source: **Deploy from branch**
3. Branch: **main** / `/ (root)`
4. Click **Save**

Website sẽ live tại: `https://YOUR_USERNAME.github.io/tdk379-website/`
